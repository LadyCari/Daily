# Daily - ParejaApp 💑

**Daily** es una aplicación multiplataforma diseñada para ayudar a las parejas a gestionar su vida cotidiana de manera organizada y colaborativa. Construida con Ionic y Angular, permite llevar un control detallado de diversas actividades desde la palma de tu mano.

## 🚀 Características Principales

- **Gestión de Gastos:** Controla los gastos individuales y compartidos para mantener las finanzas en orden.
- **Formularios Dinámicos:** Crea y personaliza tus propios formularios para registrar cualquier tipo de dato (mangas, colecciones, etc.).
- **Lista de Compras:** Gestiona tiendas, productos y listas de compras compartidas.
- **Sincronización en la Nube:** Sincroniza tus datos entre dispositivos utilizando grupos y servicios en la nube (Firebase/Google Drive).
- **Perfiles Personalizados:** Configura tu perfil y el de tu pareja para una experiencia personalizada.

## 🛠️ Stack Tecnológico

- **Framework:** [Ionic Framework](https://ionicframework.com/) con [Angular](https://angular.io/).
- **Plataforma:** [Capacitor](https://capacitorjs.com/) para despliegue en Android e iOS.
- **Iconos:** [FontAwesome Architect](https://fontawesome.com/) e [Ionicons](https://ionic.io/ionicons).
- **Backend/Storage:** Firebase y sincronización personalizada.
